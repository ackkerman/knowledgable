Knowledgable は、ドキュメントを取り込み知識グラフを生成するブラウザアプリです。主な構成要素は以下の通りです。

- **Next.js** + **tailwindcss** + **shadcn** + **GSAP** による UI
- **cytoscape.js** を用いたグラフ描画
- **turborepo** による monorepo 管理
- ブラウザローカルストレージによるデータ保存

詳しい背景や目的は [導入](./introduction.md) を参照してください。

## テスト実行方法

```bash
pnpm test
```

Node.js 標準の `node:test` を利用しています。
